# V2Ray Config Backup

This repository automatically backs up the latest V2Ray configuration file every 6 hours.

## What it does
- Fetches the latest V2Ray config
- Saves it to `server.txt`
- Commits and pushes to this GitHub repository

## Automation
Updates happen **every 6 hours** using a cron job on an Ubuntu server.

## Disclaimer
This repository is for personal backup purposes only. Use responsibly.
